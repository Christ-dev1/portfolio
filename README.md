# Portfolio — Prototype

Ceci est un prototype de portfolio professionnel créé rapidement.

Structure:
- `index.html` — page principale
- `styles.css` — styles complémentaires
- `script.js` — navigation, animations, formulaire
- `assets/` — ajoutez `CV.pdf` et votre photo professionnelle

Fonctionnalités incluses:
- Hero moderne avec badge de disponibilité
- Navigation sticky et détection de section active
- Sections: À propos, Compétences, Projets, Formation, Contact
- Formulaire de contact (simulation d'envoi)
- Scroll-to-top, effets hover, cartes de projets

Pour voir en local:
1. Ouvrir `index.html` dans un navigateur.

Pour un déploiement simple, vous pouvez utiliser GitHub Pages ou Netlify.
