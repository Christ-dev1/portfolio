1. Mettre le contenu collé dans la section Projects

a. On remplace le contenu actuel de app/components/Projects.tsx par la version complète des cartes projets.
Pourquoi : ton fichier actuel affiche seulement le haut de chaque carte, alors que le texte collé contient aussi les technologies, les fonctionnalités clés et les boutons Code / Démo.

b. On convertit le HTML en JSX.
Pourquoi : dans React / Next.js, on écrit className au lieu de class, et les styles inline utilisent des objets JavaScript.

c. On adapte les SVG.
Pourquoi : certains attributs SVG changent légèrement en JSX, par exemple stroke-width devient strokeWidth.

d. On vérifie le fichier avec ESLint.
Pourquoi : cela confirme que Projects.tsx ne contient plus d'erreur de syntaxe.
